# Real-Time Traffic Data Pipeline

## Overview
This project implements a real-time data engineering pipeline using Kafka, PySpark, Airflow, and PostgreSQL to process and analyze live traffic data.

## Architecture
Data Source → Kafka → PySpark → PostgreSQL → Dashboard

## Tech Stack
- Python
- Apache Kafka
- PySpark
- Apache Airflow
- PostgreSQL
- Streamlit
- Docker

## Features
- Real-time data streaming
- Scalable processing using Spark
- Automated pipeline orchestration
- Real-time dashboard visualization

## Use Case
Real-time traffic monitoring and congestion analysis.
